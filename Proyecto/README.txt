para jugar uno, ejecutar el archivo "jugador".

para que la maquina juegue, ejecutar el archivo "maquinav2".

el tablero se crea aleatoriamente partiendo de uno ordenado, para garantizar
que ese juego si tenga solucion. La "dificultad" se puede modificar en el codigo
diciendo cuantos movimientos tiene que realizar el tablero para desorganizarse.
La dificultad recomendada es de 10 o menor.

El funcionamiento de la maquina se basa en calcular la puntuacion actual del tablero
y mirar cual es el mejor movimiento a realizar en base a las nuevas puntuaciones.
